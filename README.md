# Tailfw
**Tailfw.exe | tail -f (w) | A simple utility for viewing appended text in a file, similar to the 'tail -f' command on Linux**

Легковесное графическое приложение для Windows, которое отслеживает изменения в файлах в реальном времени (аналог `tail -f` в Linux).

## Возможности

- **Отслеживание файлов в реальном времени** - автоматическое отображение новых строк
-  **Графический интерфейс** - удобное окно с прокруткой
-  **Настраиваемый шрифт** - выбор имени, размера и жирности шрифта через файл ```"config.ini"```
-  **Сохранение настроек** - размер и позиция окна запоминаются в файле ```"config.ini"```
-  **Поддержка UTF-8** - корректное отображение русских и других Unicode символов
-  **Обработка разных форматов** - поддерживаются `\n`, `\r\n`, `\r`
-  **Легковесность** - не требует установки, один исполняемый файл
-  **Открытый исходный код** - вы можете скачать и отредактировать файл ```tailfw.cpp``` по своему усмотрению
## Установка

1. Скачайте файл `tailfw.exe`
2. Поместите в любую удобную папку

### Запуск из командной строки
